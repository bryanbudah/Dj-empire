# 🎧 DJ Empire

A modern DJ booking and music management platform built with Django and Tailwind CSS.

## Features

- 🎵 Upload DJ mixes
- 📅 Event booking
- ⭐ Reviews & Testimonials
- 💳 M-Pesa & Stripe payments
- ☁️ Cloudinary media storage
- 👤 User authentication
- 📱 Responsive design
- 🔌 REST API
- 🚀 Deployable on Render

## Tech Stack

- Django
- Tailwind CSS
- JavaScript
- PostgreSQL
- Django REST Framework
- Cloudinary
- Docker
- GitHub Actions

## Installation

```bash
git clone <repository-url>
cd dj_website

python -m venv venv

source venv/Scripts/activate  # Windows Git Bash

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
```
